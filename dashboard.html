Overview:

I have read a couple of books on value investment recently. They led me to think that I need a comprehensive and systematic analysis framework to find promising companies. A book titled "The Warren Buffett Way" by Robert G. Hagstrom was a great starting point to dig deep into companies. So I have decided to take over this project to create a dashboard for key financial metrics measured from companies' financial statements. It's more accurate to compare companies among their peers. Therefore, I created separate dashboards for general merchandisers, tech companies, and biotech companies. This project consists of two parts. First, I wrote some Python code to prepare the datasets, namely the key financial metrics. Then, I extracted the datasets into CSV files and created the dashboards.
